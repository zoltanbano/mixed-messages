## Mixed Messages

## General Information
This is a simple node.js program that prints out a random message every time the user runs it. First, it chooses a topic, then a random element in that topic to print.

## Technologies used

- JavaSctript - ES6
- Node.js - 14.17.3
- Git - 2.32

## Setup

Download and run with GitBash/Node.
